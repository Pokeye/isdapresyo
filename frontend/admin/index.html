Predictions

What this does
Recomputes and saves forecasted fish prices using the latest stored fish price history (+ gas reference when available). This does not edit your raw fish/gas records — it only updates the generated predictions. Useful for black-box testing: different groups can change inputs (fish/gas), run again, and compare outputs.
